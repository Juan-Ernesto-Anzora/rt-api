# Sprint 1 Demo Script

## 1. Start services
- Start Docker infra
- Start API
- Start Web

## 2. Login
- Open web app
- Select tenant ACME
- Login with demo user

## 3. Home Dashboard
- Show KPI cards
- Show My Tasks
- Show Other Tasks
- Use quick filters

- Open or create a request
- Add a comment
- Attach multiple files
- Confirm one comment contains multiple attachments

## 5. Search
- Search by request title
- Search by comment text
- Search by attachment filename
- Filter by open/closed/status/priority

## 6. Logout
- Open user menu
- Logout
